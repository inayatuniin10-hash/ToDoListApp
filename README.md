# ToDoListApp
Aplikasi to-do list adalah perangkat lunak produktivitas yang dirancang untuk membantu pengguna mencatat, mengatur, dan memprioritaskan daftar tugas harian. Aplikasi ini berfungsi sebagai pengingat aktivitas agar pekerjaan lebih sistematis, membantu pengguna tetap fokus, dan meminimalkan stres akibat lupa.
